== Description ==

Vendia is a modern, clean, and performance-focused WooCommerce WordPress theme designed for product-first stores and long-term maintainability.

The theme strictly follows WordPress.org Theme Review guidelines by maintaining a clear separation between presentation and functionality. Vendia does not introduce custom post types, shortcodes, or data-locking features. All store data remains fully portable through WooCommerce and WordPress core.

Vendia provides a customizable homepage layout using the native WordPress Customizer, allowing users to control visual content such as hero text, imagery, and product sections without relying on page builders or custom admin panels.

The theme is ideal for WooCommerce stores that value simplicity, stability, and compliance with WordPress best practices.

== Features ==

* WooCommerce-ready design
* Customizer-controlled homepage sections
* Premium hero section with editable background image
* Product of the Week highlight (existing WooCommerce product)
* Featured products section
* Customer review display using WooCommerce data
* Fully responsive layout
* Accessibility-ready markup
* Editor styles included
* Translation-ready
* No external dependencies
* GPL compliant

== Homepage Customization ==

Vendiaâ€™s homepage layout is configurable through the WordPress Customizer.

Available options include:

* Hero title, subtitle, button text, and button link
* Hero background image
* Homepage products section title and product count
* Product of the Week (select by existing product ID)
* Featured products section title and product count
* Customer reviews section enable/disable and title

All homepage settings control presentation only. No content is created or stored by the theme.

== WooCommerce Compatibility ==

Vendia integrates seamlessly with WooCommerce and supports:

* Shop page
* Single product pages
* Cart and checkout
* My Account pages
* Product ratings and reviews

WooCommerce remains fully responsible for products, reviews, and store functionality.

== Installation ==

1. Upload the Vendia theme folder to /wp-content/themes/
2. Activate the theme via Appearance > Themes
3. Install and activate WooCommerce
4. Configure homepage sections via Appearance > Customize
5. Assign menus to Header and Footer locations

== Accessibility ==

Vendia follows accessibility best practices including:

* Semantic HTML5 markup
* Screen reader text support
* Keyboard accessible navigation
* Proper heading hierarchy
* ARIA attributes where required

== Frequently Asked Questions ==

= Does Vendia include demo content? =
No. The theme is designed to work with standard WordPress and WooCommerce sample data.

= Does Vendia require additional plugins? =
WooCommerce is recommended but not required.

= Is Vendia a block theme? =
No. Vendia is a classic theme designed for maximum compatibility and stability.
